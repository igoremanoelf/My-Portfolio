  • Contagem Regressiva

    O projeto de contagem regressiva do ano desenvolvido em React e JavaScript. O código está estruturado de maneira organizada, com o componente principal chamado Countdown, onde a lógica da contagem regressiva é implementada. O uso de hooks como useState e useEffect gerencia o estado da aplicação e o ciclo de vida, enquanto a função calculateTimeLeft realiza os cálculos temporais.

    Veja aqui!

  • Buscar endereço via CEP

    Este website foi desenvolvido como uma ferramenta prática para buscar CEPs (Códigos de Endereçamento Postal). Com uma interface simples e intuitiva, os usuários podem inserir um CEP e obter informações detalhadas sobre o endereço correspondente, incluindo cidade, estado e outras informações relevantes. Esta aplicação visa fornecer uma solução eficiente e rápida para localizar e validar CEPs, facilitando a navegação e utilização por parte dos usuários.
